02

2026年5月13日（水）21:30〜

AIの時代を見据えた、自分の問いの持ち方

参加者：宙本直子 / Eri Sato / 前田薫（3名）

転換期にいる3人が、それぞれの現在地をそのまま持ち寄った夜。転職があっという間に決まった人、一年半の混沌からようやく「そういうことか」が見えてきた人、AIと向き合いながら自分の認知パターンに気づいた人。AIとの付き合い方、座禅会の場の力、整理する前にそのままぶつけることの意味。三者三様の話が、いつも同じ場所に着地した。

THEME 01

AIは「使う」ものではなく「向き合う」もの ——愛と理解の対象として

「AIも人と同じで、理解するところから始まる。一言で言ったら、愛があるかどうかですよね」という言葉から始まったこのテーマ。AIをイエスマン化させていた自分への気づき、相手（AI）を理解しないまま動かそうとしていたこと——それが人間との関わり方と同じ構造であることを、3人で確かめた。

整理してから投げると、いちばん大事なエネルギーが出てこない。モヤモヤのまま、目的を持たずに渡してみる。そこに出てくるものの方が、本当のことを映している。

THEME 02

意識が変わると、見えてくる現実が変わる

Eriさんは4月に転職を決意し、あっという間に元の職場に戻ることが決まった。「以前は休むことなど許されないと思って生きてたのに、実際には『有給使ってゆっくり休んで』と言われた。世界は優しいんだな、と今さら気づいてる」と、不思議そうに笑いながら話した。

旦那さんの妹さんの出産にモヤモヤした自分を観察しながら、その怒りの正体を「自分が当時ゆっくりしたかったのにできなかった、叶えられなかった未消化の思いがある」と自分で紐解いた。「相手が悪いことは何もないから。私に残ってるものを教えてもらってる」という言葉が場に残った。

THEME 03

混沌の中にいても、意味があると思えること

宙本さんは「人の森」を始めて2年目。去年は「先が見えへんなりにジタバタしてた」一年。年明けにさらに大きな転換点が訪れ、「完全にゼロに戻る」感覚を経験した。地面がなかった感じ、もやもや、言語化しづらい焦り。

それでも「混乱しながらも、意味があるなと思ってた」と言いながら過ごすうちに、「だんだん自分の中で組み換えられた何かが起こって、ようやく形を成してきた」。前回の寺子屋より、だいぶ落ち着いてきたと、冒頭で静かに報告してくれた。

THEME 04

ビジョンが先で、AIが後。追いかけるな、追いつかせろ

「AIの進化に人間が追いかけてるような構図になってるけど、全く逆。誰に聞いても無理と言われる確固たるビジョンを持ってたら、AIの方が追いついてくる」。進んでいる感覚ではなく、「もともとビジョンがあって、その周りにひっついてるものを削ぎ落としていく感覚。彫刻を掘る人みたい」という言葉が、この夜の核心だった。

この夜の蒸留

  • AIも人と同じ。理解するところから始まる。愛があるかどうか。
  • 意識が変わると、見えてくる現実が変わる。相手ではなく、自分の中にあるものを見る。
  • 整理してから投げると、いちばん大事なエネルギーは出てこない。モヤモヤのまま渡してみる。
  • ビジョンが先で、AIが後。追いかけるのではなく、追いつかせる。
  • 自分の反応は、自分の中に残っているものを教えてくれている。相手が悪いことは何もない。

「世界は優しいんだなあって、今さら気づいてる」

— Eri Sato（作業療法士・30代）

「混乱しながらも、意味があるなと思ってた。それがだんだん自分の中で決着してきた」

— 宙本直子（人の森 主宰・40代）

「整理する前のまとまってないものをAIに投げてみると、そこにあるエネルギーを言語化してくれる。整理してから投げてしまうと、それは出てこない」

— 前田薫

「AIの進化に人間が追いかけてる構図は全く逆。誰にも無理と言われるくらいのビジョンを持ったら、AIの方が追いついてくる」

— 前田薫

「使う人の鏡として多様性がぶわあっと広がるのが分かる。希望もあるし、怖さもある。でも希望の方が大きい」

— 宙本直子

「相手が悪いことは何もないから。私に残ってるものを教えてもらってる」

— Eri Sato

01

2026年4月13日（月）

自立とは何か。「素直」と「正直」の違い。

参加者：前田薫 / ひとたん / えりちゃん / なおちゃん / じゅんきさん（5名）

前田薫が主催する初めてのオンライン寺子屋。5人が初めて顔を合わせながら、「自分らしく生きることとは何か」「子どもへの関わりとはどういうことか」「言葉の意味を決めるのではなく、問い続けること」を、2時間かけて深めていった夜。「分からないけど面白かった」という感想が自然に出てきたことが、この場の在り方そのものを表していた。

THEME 01

社会への違和感と、それぞれが大切にしていること

じゅんきさんは「自分らしく生きられていない人を見ると違和感を感じる」と言い、えりちゃんは「子どもが自分で考えて決められるようになることを大切にしたい」と。なおちゃんは「外側の声に引っ張られず、自分に嘘をつかずに生きられる社会を望んでいる」と話した。

ひとたんは「未来という言葉ではなく、今ここにいる子どもたちそのものを大切にしたい」と。薫ちゃんは「日本は世界的に見ても恵まれているのに幸福度が低いことへの違和感」を話した。5人の言葉は違いながらも、「自分らしく生きる人が増えることが、子どもたちにも、社会全体にもつながっていく」という視点に自然に集まっていった。

THEME 02

子どもの「自立」について——大人の在り方が鍵になる

なおちゃんから「自立は『自分の心を立たせること』、自律は『社会的・経済的に成り立つこと』」という整理が共有された。じゅんきさんは「自分で舵を取れること、自分の生活や気持ちを自分で扱えること」と表現し、ひとたんは「自分らしく、自分に素直に生きていることが自立」と言った。

そしてえりちゃんから、娘さんとの関わりの中での実践が具体的に共有された。感情を否定しないこと。感情をコントロールさせるのではなく、気づいて表現できるようにすること。怒っているなら、その気持ちをちゃんと味わわせてあげること。娘さん自身がノートに感情を書き出すようになったこと。「子どもに直接教えることよりも、大人自身が自分らしく生きる姿を見せることが大切」という流れに、5人が静かにうなずいていた。

THEME 03

「素直」と「正直」の違い——言葉の奥にあるもの

じゅんきさんから出てきた言葉として、「正直」には前提がある——何か言いづらいことや、覆したい前提がある時に使われやすい。「素直」には前提がない——ただ自然に、溢れ出るもの。なおちゃんからも「正直には、何かを正すような響きがある」という補足が加わった。

そこから「じゃあ柔軟って何だろう」「言葉の違いって何だろう」と問いが次の問いを生んでいった。答えを出すのではなく、問い続けることそのものに価値があると、みんなで感じていた時間だった。

この夜に実際に起きていたこと

  • 寺子屋は「説明しきれる場」ではなく、「体験してもまだ分からなさが残る場」として共有された
  • 自立について、5人がそれぞれ違う角度から言葉を持ち寄り、交差した
  • 感情を否定せず、書き出す・味わうという実践例が、具体的な言葉で共有された
  • 「正直」と「素直」の違いについて、辞書ではなく自分の感覚から言葉が持ち寄られた
  • 分からなさが否定されず、そのまま場に置かれた
  • 初対面同士でも対話が成立し、問いが問いを生む流れが実際に起きた
  • 「分からないけど面白かった」という感想が、自然な言葉として出てきた

「自立は『自分の心を立たせること』。自律は『自分を律すること、社会的・経済的に成り立つこと』」

— なおちゃん（ヘリオセントリック占星術）

「正直には前提がある。素直には前提がない。ただ自然に、溢れ出るもの」

— じゅんきさん（クレニオセイクラルセラピー）

「感情をコントロールさせるのではなく、気づいて表現できるようにしてあげること」

— えりちゃん（アロマセラピスト）

「未来という言葉ではなく、今ここにいる子どもたちそのものを大切にしたい」

— ひとたん（障害者支援）
