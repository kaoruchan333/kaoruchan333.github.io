BODY DEFAULT

なぜ学んでも、
人は変われないのか。

今回のかおる塾で深まった核は、「学び」と「身体の当たり前」は別だということです。頭では分かっていても、負荷がかかった瞬間、身体は慣れ親しんだ反応へ戻りやすい。だから中庸思考では、感情が動いた瞬間に、書く・声に出す・価値観へ戻る実践を重ねます。

よくある誤解

頭で学ぶ、決断する、行動が変わる。これは分かりやすいけれど、強い負荷がかかった時には元の反応へ戻りやすい。

今回見えた変容の順番

日々のワークで感情・認知・身体感覚を扱い、戻る先を育てる。すると、選択の瞬間に身体の反応から変わり始める。

01 / KNOW

知識として分かる

「怒らなくていい」「自分を大切にしたい」「切り替えたい」と頭では理解している状態。ここではまだ、負荷がかかった時の身体反応は変わっていない。

02 / NOTICE

感情に気づく

不快感、怒り、悔しさ、恥、焦りに気づく。ここが変容の入口。感情に気づくことで、身体がどの当たり前へ戻ろうとしているかを見られる。

03 / WRITE

身体を使って書く

出来事、感情、身体反応、片側認知を書く。手を動かし、文字を見て、呼吸しながら言葉にすることで、理解を反復可能な実践へ変える。

04 / RETURN

価値観へ戻る

負荷がかかった時に、慣れ親しんだ反応ではなく、自分の価値観へ戻る練習をする。これが中庸思考を日常に定着させる稽古になる。

TEACHING MATERIAL

講座で伝えるなら、
この順番。

「変われないのは意思が弱いから」と言わない。まず、身体が慣れ親しんだ反応へ戻る構造を伝える。その上で、感情に気づくこと、書くこと、声に出すこと、価値観へ戻ることを、具体的なワークとして渡す。

コンフォートゾーンの移動を読む
答えを与える: 頭には入るが、負荷時の反応は変わりにくい。
問いで扱う: 本人が感情と片側認知に出会える。
書いて残す: 身体を使い、見返せる痕跡になる。
反復する: 新しい戻り先を、日常の中で育てる。

学びは頭に入る。ワークは身体が戻る先を育てる。

CORE PROTOCOL

認知が本当に動く、
問いの順番。

出来事の意味づけや支援へ早く進むのではなく、まず感情を出してもいい場を整えます。そのうえで感情を生んでいる片側認知を見つけ、プラス側ならデメリット、マイナス側ならメリットを一問だけ置き、緩みが起きた後に初めて支援を扱います。

01 / SAFETY

場づくり

本人が感情を出しても否定されない場を整える。深く聞く前に受け取る。

02 / EVENT

出来事

何が起きたのか。解釈を混ぜず、具体的な事実から始める。

03 / EMOTION

感情

怒り、悔しさ、悲しみ、恥、虚しさ。不快感の種類を言葉にする。

04 / FRAMEWORK

片側認知

感情を生んでいる「〇〇は良い / 〇〇はダメ」という前提を見つける。

05 / DIRECTION

向きを判定

プラス側か、マイナス側か。ここで問いの向きを決める。

06 / PLUS

プラス側なら

「〇〇であることのデメリットは何ですか？」と問う。

07 / MINUS

マイナス側なら

「〇〇であることのメリットは何ですか？」と問う。

08 / RELEASE

緩みを確認

片側だけの見方が緩んだか、本人の感覚とことばで確認する。

09 / INTEGRATION

支援を見る

緩んだ後で初めて、出来事に含まれる支援や次の実践を扱う。

PLUS SIDE

プラス側フレームワーク

例: 「誤解なく伝わることはいいこと」「共感されることはいいこと」

「共感されることのデメリットは何ですか？」
MINUS SIDE

マイナス側フレームワーク

例: 「誤解されることはダメ」「否定されることはダメ」

「誤解されることのメリットは何ですか？」

片側認知が緩む前に、「何を受け取ったか」「何を失ったか」「どんな支援があるか」へ進みません。それは思考や前向きな解釈へ早く進ませる可能性があるためです。

PRACTICE SCIENCE

法則は、実践で
検証し続ける。

ここで扱う「科学」は、個人の思想を正解として掲げることではありません。複数のセッション記録から観察可能なサインを抽出し、同じ問いを試し、本人の言葉・感情・次の選択に何が起きたかを記録して更新する実践体系です。

観察できるサインがあるか: 感情、身体反応、本人の中心語、発話の変化。
再現できる問いにできるか: 短い一問、反対側を見る問い、統合確認。
本人の反応で確認できるか: 感情が変わったか、選択に余白が生まれたか。
反例を残せるか: 支援を急いだ時、問いが長い時、機能しなかった条件も記録する。
NEURO / COGNITION / BODY

感情・認知・身体反応を
5つの視点で見る。

中庸思考は、感情を抑える方法ではありません。出来事に対して身体が何を危険と見たのか、過去の経験からどんな予測が走ったのか、その認知が感情と行動をどう作ったのかを見る実践です。ここでは、さや先生の一次情報から整理した5つの神経科学・認知科学モデルを、コーチング学習室の言葉に翻訳して残します。

1. 感情調整プロセスモデル

感情が出た後に蓋をするのが抑圧。感情が生まれる前の認知を見直すのが認知的再評価。中庸思考は「怒りを抑える」より先に、「その怒りはどんな認知から生まれたのか」を見る。

2. 認知的再評価と前頭前野 / 扁桃体

扁桃体は素早く危険を知らせ、前頭前野は「本当に危険か」を見直す。目指すのは恐れを消すことではなく、恐れに真実を見てもらうこと。縄を蛇と見続ける反応から、縄を縄として見直せる状態へ育てる。

3. 神経内臓統合モデル

認知・感情・自律神経は互いに影響し合う。だから「頭でわかるだけ」では不十分だが、認知を見直す実践は身体反応にも波及しうる。認知の稽古は、身体の稽古でもある。

4. 構成主義的情動理論

脳は過去データやフレームワークをもとに未来を予測する。同じ出来事でも人によって感情が違うのは、出来事そのものではなく、過去の認知・予測・身体感覚が重なっているから。

5. 反復実践による神経可塑性

一回の気づきは単発。日常で何度も「事実と認知を分ける」「片側認知の反対側を見る」を繰り返すことで、新しい反応の回路が育つ。中庸道は、理解を反応へ変える反復実践である。

6. 書くことは、身体を使う認知実践

手で書くことは、手指の運動、視覚フィードバック、言葉の選択、呼吸を同時に使う。ワークシートは、頭で分かったことを身体へ落とす稽古として扱う。

WHY IT MATTERS

わかっているのに
変わらない理由。

「頭ではわかっているのにできない」は、意思が弱いからとは限りません。身体が過去に覚えた反応へ戻っているだけかもしれない。だから、さらに正しい言葉を足すのではなく、感情の入口から認知を見直し、日常で反復する必要があります。

根性論: 反応が起きた後に「頑張れ」「切り替えろ」と押す。
ポジティブシンキング: 不快な感情を前向きな言葉で置き換えようとする。
中庸思考: 感情が生まれた認知を見つけ、片側認知の反対側をシンプルに見る。
中庸道/天中道: その実践を日常・身体・場の中で反復し、反応そのものを育て直す。

書くことは、反省文ではない

ワークシートに書く目的は、正しい答えを出すことではありません。出来事を外に出し、感情を見える形にし、身体反応と片側認知を分けるためです。頭の中だけで考えると、いつもの認知の中でぐるぐるしやすい。

声に出すことは、確認である

声に出すと、自分の言葉を自分の耳で聞けます。そこで違和感が出たり、しっくりきたりする。これはコーチングで「クライアントが自分の言葉を聞いて気づく」と同じ構造です。

このセクションは医療的・科学的な断定ではなく、コーチング実践を理解するための実践モデルとして扱います。公開文では「脳科学的に証明されています」のような言い切りではなく、「こう見ると理解しやすい」「この構造で扱える」という言葉で届けます。

COACH'S CHECK

コーチ自身が
見落とさないために。

感情を聞く前に、感情を出してもいい場を作れているか。
コーチ自身が焦り・正解探し・良いセッションへの期待に入っていないか。
出来事からすぐ構造へ進まず、クライアントの感情を聞いたか。
クライアントが整理モード・説明モードに入っているサインを見ているか。
質問を一度に重ねず、一つの問いとして渡せているか。
具体の出来事と、抽象化した前提を混同せず確認できているか。
自分の見立てに答えてもらう時間ではなく、本人の言葉が増える場になっているか。

ANONYMIZED PRACTICE

実践から抽出した
二つの学び。

ここでは個人が特定される情報を公開せず、セッションから再利用できる学びのみを記録します。教材は、実際の対話に根ざしながら、誰もが扱える構造として残します。

望んだ結果が得られなかった場面

努力や手間をかけたのに結果が得られず「損した」と感じた場面では、得たものを急いで探す前に、「結果が出るべき」という前提と、その時の感情を扱う必要がある。

同調されなかった場面

「共感されることはいいこと」というプラス側フレームワークなら、「共感されることのデメリットは何ですか？」と短く問う。

学んだのに変われない場面

何度も学んだのに同じ反応へ戻る時は、「できない自分」を責める前に、身体がどの当たり前へ戻ろうとしているかを見る。そこからワークの必要性が立ち上がる。

感情に気づき始めた場面

まだ大きな行動変化がなくても、「今、感情が動いた」と気づけることは大きな変容の入口。スルーしていた反応に止まれるだけで、次の問いが置ける。
