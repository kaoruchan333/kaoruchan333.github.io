SOCCER COACHING WORKSHEET

サッカー親と選手のための
「書いて戻る」ワークシート

このワークは、頭の中で考えるだけで終わらせないためのものです。ペンを持ち、手で書き、書いた言葉を見て、できれば声に出して読む。身体を使って言葉にすることで、出来事・感情・認知・次の行動を整理しやすくします。

なぜ、手で書くのか

手書きは、視覚、手指の運動、紙やペンからの感覚、言葉の選択を同時に使います。2024年の高密度EEG研究では、大学生が手書きした時、タイピングより広い脳内結合パターンが観察されました。一方で、この研究は直接の学習テストを含まないため、「手書きなら必ず学習効果が何倍」とまでは言えません。

また、手書きノートはタイピングより逐語的な記録になりにくく、内容を要約しながら書くことで深い処理を促す可能性がある、という研究もあります。だからこのワークでは、速く入力するより、ゆっくり書いて、自分の言葉を見ることを重視します。

サッカーのメンタルは、気合いではなく、認知と身体の戻り先を育てる稽古です。

使い方

保護者へ

子どもを変えるためではなく、まず自分の見方を整えるために使います。試合後すぐではなく、帰宅後や翌日に書いても大丈夫です。

選手へ

反省文ではありません。ミスや悔しさを、次の練習で戻れる基準点に変えるためのノートです。

参考: Van der Weel & Van der Meer (2024), Frontiers in Psychology / Pinet & Longcamp (2025) commentary / Mueller & Oppenheimer (2014), Psychological Science.
WORK 1 / PARENT

保護者のための
試合後セルフコーチング

子どもに声をかける前に、親自身の感情と見方を一度紙に置きます。親の焦りが少し緩むと、子どもが話しても黙っても大丈夫な場を作りやすくなります。

1. 何が起きましたか

解釈を混ぜず、見えた出来事だけを書きます。

2. 親である自分には、どんな感情が出ましたか

悔しい焦る 不安怒り 恥ずかしいがっかり 心配期待

3. その感情の奥に、どんな前提がありましたか

例: 活躍することはいいこと / ミスすることはダメ / 黙ることはダメ / 試合に出ることはいいこと

4. 反対側を一つだけ見ます

プラス側なら

「〇〇であることのデメリットは何ですか？」

マイナス側なら

「〇〇であることのメリットは何ですか？」

5. 次の帰り道で、手放すことを一つ

評価を急がない、質問を減らす、まず黙って隣にいる、など。

WORK 2 / PLAYER

選手のための
ミス後に戻るワーク

ミスをなくすためではなく、ミスした後に自分の武器へ戻るためのワークです。弱さを探すのではなく、次のプレーに入る身体の基準点を作ります。

1. 試合中、身体が固まった場面

どの場面で、何が起きましたか。

2. その時の身体

足が止まった呼吸が浅くなった 顔が下がった視野が狭くなった 声が出なかった体が重くなった

3. その時、何が怖かったですか

例: 怒られる / 外される / 親ががっかりする / 自分には才能がないと思う

4. ミスを何として見ていましたか

例: 終わり / ダメなこと / 評価が下がること / 恥ずかしいこと

5. 次に戻る身体の基準点

利き足の小指の前、ボールの置き所、呼吸、顔を上げる前の姿勢、ミス後の一歩。

6. 次の練習で一つだけ反復すること

WORK 3 / COACH

指導者のための
言葉を足しすぎないチェック

選手が同じ反応へ戻る時、理解していないのではなく、身体が同じ安全反応へ戻っている可能性があります。言葉を増やす前に、戻れる基準点を見ます。

1. 選手に何を変えてほしいと思いましたか

2. 自分の中にあった焦り

早く変わってほしい勝たせたい 理解してほしい同じミスを見たくない 自分の指導が問われる保護者の目が気になる

3. よく言いたくなる言葉

例: 周りを見ろ / 判断を早くしろ / 切り替えろ / 自信を持て

4. 身体の基準へ翻訳すると

例: 顔が上がる置き所 / 利き足のポイント / 次の一歩 / ミス後の関わり直し

5. 次の練習で設計する反復
