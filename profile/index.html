この活動をしている理由。

奈良県天川村を拠点に、問いと対話の文化を育てながら、一人ひとりが本来に戻っていく伴走をしています。

なぜこの活動をしているのか

ビジョンは「自分らしく生きれない人をゼロにする」。個人の苦しさの背景にある認知や場の空気まで見つめ、本来へ戻る流れを作りたいと思っています。

天川村との関係

天川村の自然と静けさは、思想ではなく体感として人を本来へ戻してくれる場所です。寺子屋構想も、この土地の力とつながっています。

中庸思考との出会い

良い悪いを急いで決めるのではなく、事実と認知を分けて見ていく。その視点が、自分自身にも他者にも、ジャッジを減らしてくれました。

今のビジョン

個人だけでなく、コミュニティや組織も本来に戻していく。使命と経営を分けず、混ぜず、つなぎ直していくホームを育てています。

PERSON / PATH / PLACE

人、道、場が重なったところから、今の活動ができています。

人としての前田薫: 問いと対話の中で、その人の本来性を見つめる
道としての天中道: 価値観の言語化と中庸思考と身体実践をつなぐ
場としての天川村: 思考ではなく体感で本来に戻る理由になる
MISSION

届けたい相手

すでに社会的にも経済的にも循環している。でもどこかで、納得感や幸せ感や使命感をもう一段深く見つめたい。そんな経営者やリーダー、場を持つ人へ届けたいと思っています。

CHANGE

関わると起きていくこと

ジャッジがほどけ、自己理解と他者理解が深まり、ビジョンが明確になる。結果として、本当にやりたいことへ自然に集中できるようになっていく。その変化を大切にしています。
