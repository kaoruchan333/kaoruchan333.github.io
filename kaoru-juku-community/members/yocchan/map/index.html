現在見えている大きなテーマ

Core Theme
人からどう見られるか、役に立てているか、自分には語れるものがあるか。

よっちゃんの中では、「ちゃんとやりたい」「見てもらいたい」「自分のことを言葉にしたい」という願いが、いろいろな場面で形を変えて出てきています。

2026.04.26

「気になる」の奥にあった、見てほしい気持ち

他の人が言われていることを自分ごとのように受け取り、「自分もそう言われるかもしれない」と反応していた回。

  • 繰り返し出た言葉: 気になる、怖い、ビクビク、見てほしい
  • 見えた認知: 相手の反応や言葉で、自分が認められているかを測りやすい
  • 育った力: 「相手のことばかり気にしている」と自分で気づけた
2026.05.10

「なんかしなきゃ」が出る場面

話すことがない、役に立っていないかもしれない、見ている人にとって意味がないかもしれない、という感覚が出た回。

  • 繰り返し出た言葉: なんかしなきゃ、役に立たなきゃ、ちゃんとしなきゃ
  • 見えた認知: その場にいるなら、何か価値を出さなければならない
  • 育った力: 身体に出るサインとして「なんかしなきゃ」を見つけた
2026.05.17

「つまらない自分」の奥にある願い

自然農イベントでの自己紹介をきっかけに、「自分には言うことがない」「ただの会社員」と感じていた回。

  • 繰り返し出た言葉: 進んでいない、めんどくさい、つまらない、羨ましい
  • 見えた認知: 語れるものがある人、好きや得意を仕事にしている人は進んでいるように見える
  • 育った力: 「好きな仕事に就けているように見える」と、事実と認知を分けて言葉にできた

繰り返し出ている認知パターン

1. 人からどう見られているか

相手の言葉、反応、関心によって、自分がちゃんとできているかを確認しやすい。

2. 役に立たなきゃ

場にいるなら、何か意味のあることを出さなければならないという圧が出やすい。

3. 自分には語れるものがない

実際には経験や行動があるのに、自分の価値として認識しきれていない。

4. 羨ましさに願いが映る

好きなこと、得意なこと、自分の活動を語れる人に反応する。そこには「自分もそうありたい」という願いがある。

育っている力

  • 感情が動いたことに気づく力
  • 自分の認知を「事実」ではなく「そう見えている」と分ける力
  • 何もないと思うところから、実際に起きている出来事を拾う力
  • 羨ましさや怖さを、悪いものにせず入口として見る力

次に見ていく問い

  1. 「自分には言うことがない」と思う時、事実としては何をしている人なのか。
  2. 「役に立たなきゃ」が出る時、誰にどう見られることを守っているのか。
  3. 羨ましい人の中に、自分のどんな願いを見ているのか。
  4. 仕事の中で、自分がすでに人の困りごとを軽くしている部分はどこか。
