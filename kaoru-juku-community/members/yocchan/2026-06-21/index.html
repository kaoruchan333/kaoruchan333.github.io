「報われなかった」の奥にあった、見てもらえなかった3年前

昇進そのものではなく、我慢してやり切った時間が見てもらえなかった痛みを、事実と認知に分けて見る。

2026年6月21日 かおる塾グループコーチング

1. タイトル

Cognitive Structure Sheet
よっちゃん 認知構造分析シート
「報われなかった」は本当ですか

2. はじめに

このページは、よっちゃんを評価したり、会社の出来事を外側から決めつけたりするものではありません。

今回大切だったのは、「反対側を見る」より前に、よっちゃん自身の中に残っていた「悲しい」「悔しい」「理不尽」「見てもらえなかった」という片側を、言葉として出せたことです。

そしてもう一つ大きかったのは、よっちゃん自身が途中で「変な認知だ」と気づいていたことです。認知が悪いのではありません。事実と認知がひっついていると苦しくなる。だから、分けて見ていく。そのための振り返りです。

3. 今回扱った出来事・テーマ

  • 同じ会社の隣で働く人が昇進するらしいと聞いた。
  • よっちゃんは「あ、また上がらんかったな」と感じた。
  • 「悲しい」「悔しい」という言葉が出た。
  • 「見てもらえなかった」「我慢してやってきたのに」「いつか頑張りを見てくれるかなって待ってた」という言葉が出た。
  • 「長年の我慢してきたのが、いつか報われたい」と言葉になった。
  • 「外に出された」「もう見てもらえない」「もう気にもしてもらえないような感じ」と感じていた。
  • 最後に、3年前の大変な時に見てもらえなかったこと、そこが本当に嫌だった場所として見えてきた。

※このページでは、会社の評価や昇進の是非を事実認定しません。ここで扱うのは、よっちゃんの内側で起きていた認知と感情です。

4. 事実と認知の整理

事実よっちゃんの認知・解釈感情奥にありそうな本音
同じ会社の人が昇進するらしいと聞いた。また自分は上がらなかった。悲しい、悔しい。自分も見てほしかった。
過去に大変な仕事をやり切った。我慢してやってきたのに、見てもらえなかった。悔しさ、理不尽さ。あの時の自分を分かってほしかった。
今は以前とは違う立場・職場にいる。外に出された。もう見てもらえない。気にもしてもらえない。寂しさ、怒り、置いていかれた感覚。必要だと言われたことが、本当であってほしかった。
今後昇進したとしても、時期は過去とは違う。今更。年数だったからと思いそう。虚しさ。3年前のあの時に、見てほしかった。

5. 繰り返し出てきた言葉

また上がらんかった / 悲しい / 悔しい / 見てもらえなかった / 我慢してやってきたのに / いつか報われたい / 外に出された / もう見てもらえない / いいように使われた / 口だけ / 今更 / 理不尽 / 不公平

今回の言葉は、かなりはっきりしていました。よっちゃんが嫌だったのは、単に「昇進しなかったこと」だけではなさそうです。

もっと深いところには、「あの大変だった時の自分を見てほしかった」「我慢してやり切った時間が、なかったことみたいになってほしくなかった」という痛みがあったように見えます。

6. 表層のテーマ

表面のテーマは、昇進や会社での評価でした。

ただ、セッションの中で見えてきたのは、「昇進したいかどうか」よりも、「見てもらえなかった」「報われなかった」という認知が、過去のある一点に強く結びついていることでした。

7. 深層のテーマ

深いテーマは、よっちゃんが、我慢してやり切った自分をどう扱っているかです。

「あの時、あれだけやった」「精神的にも、仕事優先で、いろいろ犠牲にして、なんとかやり切った」。そこまでした自分が、3年前のタイミングで見てもらえなかった。

今回の出来事は、その痛みにもう一度触れた入口だった可能性があります。

今回の昇進が嫌だった、というより、3年前に見てもらえなかった痛みがまだ残っていた。

8. 認知のフレームワーク

今回見えていた可能性のあるフレーム

  • 我慢してやり切ったら、いつか報われるはず。
  • 必要だと言われたなら、ちゃんと見てもらえるはず。
  • 大変だった時の頑張りを忘れられるのは、つらい。
  • あの時に見てもらえなかったなら、後から評価されても今更に感じる。
  • 報われたかどうかを、ひとつの出来事・ひとつのタイミングで測ってしまう。

ここで大事なのは、このフレームが悪いということではありません。むしろ、よっちゃんがそれだけ長い間、仕事にエネルギーを注いできたからこそ出てくるものです。

ただ、このフレームだけで見ると、「報われたかどうか」が一つの出来事に閉じ込められてしまいます。

9. 欠落感・価値観・才能との接続

よっちゃんの才能は、見えにくい穴や負荷を察知し、できない状態をできる形へ整えていくところにあります。

その力は、仕事の中で大きく使われてきたはずです。ただ、その力が「我慢してでもやり切る」「穴を埋める」「必要とされたら応える」方向へ使われ続けると、あとから「私のことは見てもらえなかった」という痛みに変わることがあります。

だから今回見るべきは、「昇進できたかどうか」だけではありません。よっちゃんが実際に何を整え、何を守り、誰を助け、どんな力を使ってきたのか。その事実です。

10. よっちゃんへの問い

  1. 「報われなかった」と感じる時、いちばん思い出す場面はどこですか。
  2. その場面で、よっちゃんは実際に何をしていましたか。
  3. その時、誰の仕事・誰の不安・どんな問題を軽くしていましたか。
  4. 「犠牲にした」と感じるものを、具体的に書くと何ですか。
  5. その犠牲の中で、よっちゃんが守っていたものは何ですか。
  6. 「見てもらえなかった」は認知だとして、事実として見えていたもの・助けられていたものはありますか。
  7. 「報われなかった」という言葉を、今すぐ消さなくていいとしたら、その横に置ける事実は何ですか。

11. かおるちゃんから

よっちゃん、今回すごく大きかったのは、「見てもらえなかった」「我慢してやってきたのに」「いつか報われたい」という言葉が、よっちゃん自身の口から出てきたことだと思っています。

これは、反対側を見られていないという話ではありません。むしろ、反対側を見る前に、本当に嫌だった場所へようやく近づけたということです。

「報われなかった」は、今のよっちゃんにとって強く本当のように感じる認知です。だから否定しません。ただ、その言葉だけで過去の全部を閉じ込めなくてもいい。

あの時、よっちゃんは実際に何をしていたのか。誰を支えていたのか。何を整えていたのか。そこに事実として、すでに受け取っていたものは何か。

そこを、急がず一緒に見ていきましょう。
